# 思源笔记“搜easy”插件版本更新记录

## v1.2.0
### 新增
- 增加大代码块搜索性能优化设置开关，改善>1000行大代码块搜索卡顿问题；
- 支持文档锁定状态下的搜索命中高亮和跳转；

### 优化
- 改善大文档延迟加载导致搜索命中关键词无法成功跳转问题；
- 修复同一块搜索命中顺序与实际内容不一致问题；
- 修复搜索命中关键词在折叠列表项，导致无法跳转问题；
- UI细节优化，改进大文档搜索体验；

## v1.0.0
搜 easy 为思源笔记当前文档提供更接近 VS Code 的查找与替换体验。

### 首版本功能
- 在当前文档内逐个查找与替换
- 区分大小写、全词匹配、正则匹配
- 支持仅在当前选区内查找与替换
- 支持当前文档 minimap 与命中位置显示
