# 搜 easy

搜 easy 为思源笔记当前文档提供更接近 VS Code 的查找与替换体验。

## 功能

- 在当前文档内逐个查找与替换
- 上一项 / 下一项导航与实时命中计数
- 区分大小写、全词匹配、正则匹配
- 支持仅在当前选区内查找与替换
- 支持替换当前、跳过当前、全部替换
- 按当前命中的大小写形式调整替换文本
- 支持顶栏入口、命令面板入口和可配置快捷键
- 支持面板拖拽、缩放与位置记忆
- 支持当前文档 minimap 与命中标记联动

## 平台支持

- 已支持：`desktop`、`desktop-window`、`browser-desktop`
- 暂不支持：移动端前端

## 使用边界与风险提示

- 仅作用于当前激活文档
- 查找与定位依赖当前编辑器 DOM
- 遇到复杂富文本结构时，部分命中可能只能查找，暂不适合直接替换
- 对重要文档执行“全部替换”前，建议先逐项确认命中结果

## 开发

```bash
pnpm install
pnpm test
pnpm precheck
pnpm build
```

如果 `.env` 已配置 `VITE_SIYUAN_WORKSPACE_PATH`，开发时还可以使用：

```bash
pnpm dev
```
