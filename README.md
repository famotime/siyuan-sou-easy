# siyuan-friendly-search-replace

A SiYuan plugin that adds a more editor-friendly search-and-replace flow for the current document.

- Search by text matches instead of a block-first mental model
- Navigate matches one by one
- Replace current, skip current, or replace all
- Preserve replacement case for lower / upper / title-cased matches
- Inspired by the in-editor find/replace workflow in VS Code

## Development

```bash
npm install
npm run test
npm run build
```
