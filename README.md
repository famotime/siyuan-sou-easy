# 搜 easy

搜 easy 为思源笔记当前文档提供更接近 VS Code 的查找与替换体验。

## 功能

- 在当前文档内逐个查找与替换
- 区分大小写、全词匹配、正则匹配
- 支持仅在当前选区内查找与替换
- 支持当前文档 minimap 与命中位置显示

## 使用边界与风险提示

- 仅作用于当前激活文档
- 查找与定位依赖当前编辑器 DOM
- 遇到复杂富文本结构时，部分命中可能只能查找，暂不适合直接替换
- 对重要文档执行“全部替换”前，建议先逐项确认命中结果

## 重要说明

本插件涉及文档内容编辑，此类变更无法使用编辑回退，只能通过思源笔记的“数据历史”恢复，存在一定风险。

本插件在发布前虽已经过开发者本人试用验证，但仍可能有不可预知的Bug，建议不要在重要文档使用。
